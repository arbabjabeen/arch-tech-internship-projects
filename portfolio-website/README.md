# Arbab Jabeen — Personal Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, experience, and achievements. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks or external APIs.

## ✨ Features

- **Modern Dark Theme** — Professional dark navy palette with teal accent highlights
- **Fully Responsive** — Mobile-first design with hamburger menu, adapts to all screen sizes
- **Smooth Animations** — Scroll-reveal fade-ins, card hover lift effects, and button transitions
- **Interactive Navigation** — Smooth scroll to sections with active link highlighting
- **Contact Form** — Frontend validation with inline error messages and success feedback
- **7 Sections** — Hero, About, Skills, Projects, Experience, Achievements, Contact

## 🛠️ Technologies Used

| Technology  | Purpose                          |
|-------------|----------------------------------|
| HTML5       | Semantic page structure          |
| CSS3        | Styling, layout, animations     |
| JavaScript  | Interactivity and form handling  |

## 🚀 How to Run

1. Download or clone the project folder.
2. Open `index.html` in any modern web browser.
3. No installation, build step, or server required.

## 📐 Sections Overview

1. **Hero** — Name, title, subtitle, intro, CTA buttons
2. **About Me** — Biography, education, career ambition
3. **Skills** — Technical & soft skills in card grid
4. **Projects** — 3 project cards with tech tags
5. **Experience** — Vertical timeline layout
6. **Achievements** — Academic, certifications, hackathons
7. **Contact** — Info links + validated contact form
